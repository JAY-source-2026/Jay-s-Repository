03

내화채움구조 — 관통부의 빈틈이 화재의 통로가 됩니다

FIRESTOP
SHEET 03 / 08
01 평상시벽·바닥 관통부에 자재를 밀실하게 충전한 상태로 방화구획을 유지합니다.
개요 · DESCRIPTION

벽과 바닥을 지나는 배관·케이블·덕트 주변의 틈은 화염과 유독가스가 퍼지는 길이 됩니다. 디오이엔씨의 내화채움구조는 이 관통부를 빈틈없이 채워 방화구획의 성능을 유지합니다.

사양 · SPECIFICATION
차단 원리
화재 시 발포하여 탄화층을 생성, 화염과 유독가스를 차단

관련 특허 다수 보유, 자체기술 국산화 제품

04

셔터 — 넓은 공간일수록 구획이 필요합니다

SHUTTER
SHEET 04 / 08
01 평상시셔터는 상부에 수납되어 있고, 대공간은 그대로 열린 채 사용됩니다.
개요 · DESCRIPTION

대형 매장, 물류창고, 주차장처럼 트인 공간은 화재가 빠르게 번집니다. 방화셔터와 스크린셔터는 화재 감지 시 자동으로 내려와 공간을 방화구획으로 나누고 피난 시간을 확보합니다.

사양 · SPECIFICATION
특허
스크린셔터 특허 보유 — 대공간 구획 대응
작동
화재 감지 설비와 연동한 자동 폐쇄
설계
피난 동선을 고려한 설계 및 시공
제작
현장 규격에 맞춘 자체 생산 맞춤 제작
적용처
대형 매장 · 물류창고 · 주차장 등 대공간
주요 치수 · 요청 항목
W × Hmm

개구부 실측 치수. 현장 규격에 맞춰 자체 생산합니다.

06

ONE-STOP 공정

PROCESS
기술개발 → 시공

자재를 사오는 회사가 아닙니다. 기술을 직접 개발하고, 자재를 직접 만들고, 설계하고, 현장에 직접 시공합니다. ONE-STOP SYSTEM이 성능의 편차를 없앱니다.

  1. 01

    기술개발

    자체 연구개발로 화재차단 기술을 국산화하고 특허를 확보합니다.

  2. 02

    자재생산

    불연성 친환경 무기 화합물 기반 자재를 직접 생산합니다.

  3. 03

    설계

    설계소프트웨어로 현장 조건에 맞는 인정구조를 선정합니다.

  4. 04

    시공

    건축물과 지하구 현장에 직접 시공하여 성능을 완성합니다.
